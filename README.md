# Renderer
A 3D renderer using the Vulkan API.
An educational playground to figure out Vulkan, 3D graphics, and various rendering techniques
![](PBRTextured.PNG)
---
## Implemented
- Boilerplate Vulkan
- Models and textures
- Normal mapping
- Phong shading
- PBR shading

## Queued
- IBL
- Shadowmapping
- Deferred shading
- SSAO
- Ray tracing
- Better modularity
